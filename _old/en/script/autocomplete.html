Auto Complete

A UI component that shows a list of keywords containing the input value when inputting a string in a text box

Default

If the input text is found among the keywords below, the keyword in question is displayed on the list.  

var ac_words = [
  "ActionScript", "AppleScript", "Asp", "BASIC", "C", "C++",
  "Clojure", "COBOL", "ColdFusion", "Erlang", "Fortran", "Groovy",
  "Haskell", "Java", "JavaScript", "Lisp", "Perl", "PHP",
  "Python", "Ruby", "Scala", "Scheme"
];

jui.ready([ "uix.autocomplete" ], function(autocomplete) {
	ac_1 = autocomplete(".ac", {
		target: "input[type=text]",
		words: ac_words,
		event: {
			change: function(text) {
				alert(text);
			}
		}
	});
});

Below is the markup and template of an autofill component. The selector is a group tag and the text box can be selected using the target option.

<span class="ac group">
	<label class="label label-small"><i class="icon-search"></i></label>
	<input type="text" class="input input-small" style="width: 100px" />
</span>

<script data-jui=".ac" data-tpl="words" type="text/template">
	<div class="dropdown">
		<ul>
		<! for(var i = 0; i < words.length; i++) { !>
			<li><!= words[i] !></li>
		<! } !>
		</ul>
	</div>
</script>

Update

Run - Changed into the word list shown below when you click the left button.  

var ac_words_hangul = [
