Dropdown

Drowndown is a UI component that is frequently used in multiple UI components such as combo box, navigation, table, ect.

Default

A basic example for the dropdown component is shown below. It is possible to specify a value for each element (only numerical characters can be entered).

  • Dropdown 1
  • Dropdown 2
  • Dropdown 3
  • Dropdown 4

jui.ready([ "ui.dropdown" ], function(dropdown) {
	dd_1 = dropdown("#dd_1", {
		event: {
			change: function(data) {
				alert(data.value + ", " + data.text);
			}
		}
	});
});
<div style="display: inline-block; position: relative;">
	<div class="group">
		<button class="btn btn-black" onclick="dd_1.show();">Show</button>
		<button class="btn btn-black" onclick="dd_1.hide();">Hide</button>
	</div>
	
	<div id="dd_1" class="dropdown dropdown-large">
		<ul style="width: 150px;">
			<li value="1">Dropdown 1</li>
			<li class="divider"></li>
			<li value="2">Dropdown 2</li>
			<li value="3">Dropdown 3</li>
			<li value="4">Dropdown 4</li>
		</ul>
	</div>
</div>

Anchor + Close

If you want to apply a speech balloon effect to a dropdown, add a tag with an anchor class to the dropdown.
In addition, if you want to place the speech balloon on the right side, add an anchor-right class to the relevant speech balloon.

If the value of the close option is set to false, the dropdown will not be closed when clicking on the list. 

dd_2 = dropdown("#dd_2", {
	close: false
});

Style + Keydown

If a dropdown is shown on the right side of the screen, a part of the menu may be blocked.
To resolve this problem, set the margin-left style value to a negative number.

It is possible to adjust the location of a dropdown by changing the left margin value when performing a right alignment. 

<div id="dd_3" class="dropdown dropdown-black" style="margin-left: -15px;">
	<!-- Omitted -->
</div>

It is possible to choose a list using the keyboard direction keys when the value of the keydown option is set to true. 

dd_3 = dropdown("#dd_3", {
	keydown: true
});

Update

The list of a dropdown can be configured not only with markup but also solely with data.

Update Menu List

Assign the selector of a specified dropdown to the data-jui property value, and assign the data-tpl property value to node. 

<script data-jui="#dd_4" data-tpl="node" type="text/template">
	<li value="<!= value !>"><!= text !></li>
</script>

If a node template is configured, data can be configured with options as follows. 

dd_4 = dropdown("#dd_4", {
	nodes: [
		{ value: 1, text: "text1" },
		{ value: 2, text: "text2" },
		{ value: 3, text: "text3" }
	]
});

The user can change the dropdown list by clicking the update button. 

dd_4.update([
	{ value: 4, text: "text4" },
	{ value: 5, text: "text5" },
	{ value: 6, text: "text6" }
]);
