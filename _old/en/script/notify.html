notify

A UI component that provides an alarm function for when a serious problem or event occurs.

Default

The following table is an example list that facilitates the testing of component(s).

Result Options
{ position: "top-right" }
{ position: "top-left", timeout: 0 }
{ position: "top", timeout: 2000, padding: { top: 100 } }
{ position: "bottom", timeout: 0, distance: 50 }
{ position: "bottom-left", showDuration: 1000, hideDuration: 1000 }
{ position: "bottom-right", showEasing: "linear" }

The following is a basic template that configures a notice. 

<script id="tpl_alarm" type="text/template">
    <div class="notify notify-<!= color !>">
        <div class="title"><!= title !></div>
        <div class="message"><!= message !></div>
    </div>
</script>

The user can add a notice message to the selector set when creating an object. 

jui.ready([ "ui.notify" ], function(notify) {
	notify1 = notify("body", {
        position: "top-right",
        event: {
        	show: function(data) {
        		console.log("show : " + JSON.stringify(data));
        	}, 
        	hide: function(data) {
        		console.log("hide : " + JSON.stringify(data));
        	},
        	click: function(data) {
        		console.log("click : " + JSON.stringify(data));
        	}
        },
        tpl: {
            item: $("#tpl_alarm").html()
        }
    });
});
