Ready

The JUI library can use a component module inside of the ready method callback. 

jui.ready(function(ui, uix, _) {
	var combo = ui.combo(selector, options),
		table = uix.table(selector, options);
		
	//...
});

The ui, uix and _object, which can be received as parameters of the callback method, provide the following functions.

Name Description
ui A component whose independence from other components is guaranteed.
uix A component that provides more complex and varied functions because it is created using combinations of other components.
_ Provides a standardized utility method that can be used inside and outside the component.

ui and uix objects, which can be received as parameters of the callback function, can create the following components.

Package Type Description
ui button Component that provides checkbox and radio button functions.
ui combo Component that implements the select box appearance.
ui dropdown Component that can display a menu-type list.
ui tooltip A visualization component that contributes to the definition of a specific area.
ui modal Component that displays the screen of a specific object more clearly.
ui paging Component that generalizes and provides the paging which is frequently used in a service such as a bulletin board or a guest book.
ui datepicker Component for selecting date/month/year.
ui notify Component that shows a notice message when a significant error or event occurs in the service.
uix window Component with a typical Windows format.
uix table Component that specializes in adding, modifying and deleting data and is able to display data in a tree structure.
uix xtable Specializes in retrieving data, unlike the table component. It can retrieve over 1 million records.
uix tab Component that can display multiple sections of content using tabs.
uix tree Component that can display hierarchical data such as directories.
uix autocomplete Component that helps with text completion when writing in a text box

If a module name is supplied in thefirst parameter , it is possible to load only the module concerned. However, the module name is usually omitted. 

jui.ready(function(ui, uix, _) {
	var combo = ui.combo(selector, options),
		table = uix.table(selector, options);
		
	//...
});

create

A method for creating a component object dynamically, even outside of the callback of the ready method. OHowever, the create method can only be used once the callback of the ready method is complete. 

var combo = jui.create("combo", selector, options),
	table = jui.create("table", selector, options);
