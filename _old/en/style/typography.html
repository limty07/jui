Heading + Text

One could use this for an auxiliary sentence of the title when also using a small tag.

h2. Jennifer UI heading Secondary text
h3. Jennifer UI heading Secondary text
h4. Jennifer UI heading Secondary text
h5. Jennifer UI heading Secondary text
h6. Jennifer UI heading Secondary text

<div class="h2">h2. Jennifer UI heading <small>Secondary Text</small></div>
<div class="h3">h3. Jennifer UI heading <small>Secondary Text</small></div>
<div class="h4">h4. Jennifer UI heading <small>Secondary Text</small></div>
<div class="h5">h5. Jennifer UI heading <small>Secondary Text</small></div>
<div class="h6">h6. Jennifer UI heading <small>Secondary Text</small></div>

Blockquote

A style that is appropriate for a sentence or phrase cited from other writings.

The worst decision of all is not making any decisions. Lucius Annaeus Seneca

The worst decision of all is not making any decisions. Lucius Annaeus Seneca
