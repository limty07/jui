Calendar

클래스 이름만 calendar로 변경하면 다른 스타일의 달력 컴포넌트를 사용할 수 있습니다.

SUNMONTUEWEDTHUFRISAT

아래는 달력 선택 컴포넌트의 템플릿이며, 요일 부분은 환경에 맞게 수정해서 사용할 수 있습니다. 

<div id="datepicker1" class="datepicker">
    <div class="head">
        <div class="prev"></div>
        <div class="title"></div>
        <div class="next"></div>

        </tr>
    </table>
</div>

아래는 달력 객체를 생성하는 코드입니다. 타이틀과 날짜 클릭시 발생되는 포맷을 옵션으로 설정할 수 있습니다. 

jui.ready([ "ui.datepicker" ], function(datepicker) {
	datepicker1 = datepicker("#datepicker1,.calendar", {
	    titleFormat: "yyyy. MM",
	    format: "yyyy/MM/dd",
	    event: {
			select: function(date, e) {
				alert(date);
			},
			prev: function(e) {
				alert("prev");
			},
			next: function(e) {
				alert("next");
			},
			tpl: {
				dates: $("#tpl_dates").html()
			}
	    }
	});
});

Monthly & Yearly

달력 선택 컴포넌트는 월과 연도를 선택할 수 있는 타입도 제공합니다.

달력의 구성 요소를 템플릿으로 분리함으로써 다국어 설정이 가능하도록 하였습니다. 

<script data-jui="#datepicker2" data-tpl="dates" type="text/template">
    <tr>
        <! for(var i = 0; i < dates.length; i++) { !>
        <td><!= dates[i] !></td>
        <! } !>
    </tr>
</script>

<script data-jui="#datepicker3" data-tpl="dates" type="text/template">
    <tr>
        <! for(var i = 0; i < dates.length; i++) { !>
        <td><!= dates[i] !></td>
        <! } !>
    </tr>
</script>

달력의 타입을 변경하기 위해서는 type 옵션을 사용하면 됩니다. 

datepicker2 = datepicker("#datepicker2", {
	type: "monthly",
    titleFormat: "yyyy",
    format: "yyyy/MM",
    event: {
    	select: function(date, e) {
    		alert(date);
    	}
    }
});

datepicker3 = datepicker("#datepicker3", {
	type: "yearly",
    format: "yyyy",
    event: {
    	select: function(date, e) {
    		alert(date);
    	}
    }
});
