Ready

JUI 라이브러리는 ready 메소드의 콜백 내부에서 컴포넌트 모듈을 사용할 수 있습니다. 

jui.ready(function(ui, uix, _) {
	var combo = ui.combo(selector, options),
		table = uix.table(selector, options);
		
	//...
});

먼저 콜백 함수의 인자로 받을 수 있는 ui와 uix 그리고 _ 객체는 다음과 같은 기능을 제공합니다.

Name Description
ui 다른 컴포넌트에 종속적이지 않는 독립성이 보장된 컴포넌트입니다.
uix 다른 컴포넌트와 조합하여 좀 더 복합적이고 다양한 기능을 제공하는 컴포넌트입니다.
_ 컴포넌트 내부와 그 밖에 외부에서도 사용할 수 있도록 정형화된 유틸 메소드를 제공합니다.

콜백 함수의 인자로 받을 수 있는 ui와 uix 객체는 다음과 같은 컴포넌트를 생성할 수 있습니다.

Package Type Description
ui button 체크 및 라디오 버튼의 기능을 제공하는 컴포넌트입니다.
ui combo 셀렉트 박스를 좀 더 시각화하여 구현한 컴포넌트입니다.
ui dropdown 메뉴 형태의 리스트를 보여줄 수 있는 컴포넌트입니다.
ui tooltip 특정 영역에 대한 설명을 좀 더 시각화하여 보여주는 컴포넌트입니다.
ui modal 특정 대상 화면을 더 뚜렷하게 보여주기 위한 컴포넌트입니다.
ui paging 게시판이나 방명록과 같은 서비스에서 자주 사용되는 페이징을 일반화하여 제공하는 컴포넌트입니다.
ui datepicker 일/월/년의 날짜를 선택할 수 있는 컴포넌트입니다.
ui notify 서비스에서 심각한 문제나 이벤트 등이 발생하였을 경우에 알림 메시지를 보여주는 컴포넌트입니다.
uix window 일반적인 윈도우 형태의 컴포넌트입니다.
uix table 데이터를 추가 및 수정, 삭제에 특화되어 있으며 트리 구조의 데이터도 표현할 수 있는 컴포넌트입니다.
uix xtable table 컴포넌트와는 달리 데이터 조회에 특화되어 있으며, 백만건 이상의 데이터를 조회할 수 있습니다.
uix tab 다수의 영역을 탭으로 분리하여 표현할 수 있는 컴포넌트입니다.
uix tree 디렉토리와 같은 계층형 데이터를 표현할 수 있는 컴포넌트입니다.
uix autocomplete 텍스트박스에 입력하였을 경우, 문장의 완성을 도와주는 컴포넌트입니다.

그리고 첫번째 매개변수에 정의된 모듈 명을 배열 형태로 설정한다면 해당 모듈만 로드하여 사용할 수 있지만 보통 생략하여 사용합니다. 

jui.ready([ "ui.combo", "uix" ], function(combo, uix) {
	var combo = combo(selector, options),
		table = uix.table(selector, options);
		
	//...
});

Create

ready 메소드의 콜백 외부에서도 컴포넌트 객체를 동적으로 생성할 수 있는 기능을 제공합니다. 한가지 주의해야 할 사항은 create 메소드는 ready 메소드의 콜백이 실행된 이후에 사용할 수 있습니다. 

var combo = jui.create("combo", selector, options),
	table = jui.create("table", selector, options);
	
//...
