X-Table

수백만건의 데이터 로우를 처리할 수 있는 대용량 테이블입니다.
기존의 성능에 문제가 있었던 이너 스크롤 문제를 해결하였으며, 버퍼를 활용한 페이징이나 스크롤링 기능을 제공합니다.

Scroll (+Dropdown, +Loading)

Run - 100만건의 데이터를 추가하는 데모이며, 스크롤이 맨 밑으로 내려갔을 경우에 로우를 추가적으로 렌더링합니다.

jui.ready([ "uix.xtable" ], function(xtable) {
	xtable_1 = xtable("#xtable_1", {
		fields: [ "name", "age", "location" ],
		resize: true,
		colshow: true,
		sort: true,
		sortLoading: true,
		buffer: "scroll", // scroll, page, s-page
		bufferCount: 20,
		event: {
			colmenu: function(column, e) {
				this.showColumnMenu(e.pageX - 25);
			}
		}
	});
});

Filter

Filter Multiple Filter - 좌측에 버튼을 클릭하면 특정 기준에 맞게 테이블의 로우가 필터링 됩니다.

Name Age Location 

function xtableFilter() {
    xtable_4.filter(function(data) {
        if(data.location.indexOf("eo") != -1) {
            return true;
        }
    });
}

function xtableFilterMulti() {
    xtable_4.filter(function(data) {
        if(data.age >= 30 || data.name.indexOf("ng") != -1) {
            return true;
        }
    });
}

Horizontal Scroll

컬럼 개수가 많거나 테이블 넓이가 클 경우에 scrollWidth 옵션을 사용하면 테이블 가독성을 높일 수 있습니다.

Name Age Location

테이블의 실제 넓이가 scrollWidth 옵션 값보다 클 경우에 가로 스크를이 보여집니다.
참고로 테이블의 실제 넓이는 width 옵션을 사용하거나 마크업의 스타일로도 설정할 수 있습니다. 

xtable_5 = xtable("#xtable_5", {
	fields: [ "name", "age", "location" ],
	data: table_data,
	resize: true,
	sort: true,
	width: 1000,
	scrollWidth: 727,
    animate: true
});
