Tree

Default

트리는 총 4가지의 스타일을 제공하며, 각각의 리스트 태그에는 상태에 따른 클래스를 정의할 수 있습니다.

Class Description Code
tree tree-arrow 화살표 형태의 심플한 스타일 <span>ROOT</span>
tree tree-arrow-file 화살표 + 파일 형태의 복합 스타일 <span><i></i> ROOT</span>
tree tree-line 라인 형태의 심플한 스타일 <div>ROOT</div>
tree tree-line-file 라인 + 파일 형태의 복합 스타일 <div><i></i> ROOT</div>

참고로 타이틀 영역의 태그는 A, SPAN, DIV 중 하나만 사용하면 됩니다. 

<ul class="tree tree-style">
	<li class="open root">
		<i></i><a href="#">ROOT</a>
		<ul>
			<li class="open">
				List 1
			</li>
			<li class="fold">
				List 2
			</li>
			<li class="leaf last">
				List 3
			</li>
		</ul>
	</li>
</ul>
