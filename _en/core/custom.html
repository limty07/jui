JUI library provides a send box method that facilitates the development of UI components.

Define

To define a new component module must set the parameters in the following order. (New module name, Using module name, Module object handler) 

jui.defineUI("ui.test", [ "util.base" ], function(_) {
	var UI = function() {
		this.init = function() {
			// initially implemented part
		}
		
		this.func1 = function(val) {
			// open method 1
		}
	}

    UI.setup = function() {
        return { // default UI option settings
			option1: 1000,
			option2: true
        }
    }
	
	return UI;
}, "core");

The UI.setup method defines the option(s) to be used in connection with a component, and init is a method that is initially run when a component object is created. These two methods must be defined when adding a new UI component.

core

In the class returned by the define method, the basic methods and properties required by a UI component can be called using this. For example, a custom method can be created with the edit method. If you are using the applicable component you can listen for a custom event using the on method or the component event option.
