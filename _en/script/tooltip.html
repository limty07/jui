Tooltip

Tooltip can be applied to 4 locations, namely top/bottom/left/right, and a relate message can be entered in the title properties. 

<span class="tooltip" title="Tooltip Message">Demo</span>

Top

Sets the location of a tooltip to the top. The tooltip is shown in response to a mouseover motion (Over). 

jui.ready([ "ui.tooltip" ], function(tooltip) {
	tooltip_1 = tooltip(".tooltip_1");
});

Bottom

Sets the location of a tooltip to the bottom. The color of a tooltip is set to gray and the tooltip is shown in response to a mouseover motion.

Tooltip Bottom 
tooltip_2 = tooltip(".tooltip_2", {
	position: "bottom",
	color: "gray",
    showType: "click"
});

Left

Sets the location of a tooltip to the left. The tooltip is shown 1 second after the mouseover motion (Delay). 

tooltip_3 = tooltip(".tooltip_3", {
	position: "left",
	delay: 1000
});

Right

Sets the location of a tooltip to the right side and sets the max-width value to 300px. If the tooltip line reaches the end, it automatically breaks the line.
The tooltip is center-aligned (Over). 

tooltip_4 = tooltip(".tooltip_4", {
	position: "right",
	width: 300,
	align: "center"
});

With Popover

Tooltips can be used with pop-over style. (Over) 

tooltip_5 = tooltip(".tooltip_5", {
    position: "right",
    width: 300,
    align: "left",
    tpl: {
        item: $("#tpl_popover").html()
    }
});

To use the pop-over style, you must set the item template. 

<script id="tpl_popover" type="text/template">
    <div class="popover popover-<!= position !> popover-<!= color !>">
        <div class="head">Subject</div>
        <div class="body">
            <div class="image"><i class="icon-caution"></i></div>
            <div class="message"><!= message !></div>
        </div>
    </div>
</script>
