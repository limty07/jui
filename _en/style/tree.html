Tree

Default

Tree provides a total of 4 styles and, for each list tag, it is possible to define a class in accordance with the state.

Class Description Code
tree tree-arrow Simple style in an arrow shape <span>ROOT</span>
tree tree-arrow-file Composite style in an arrow + file shape <span><i></i> ROOT</span>
tree tree-line Simple style in a line shape <div>ROOT</div>
tree tree-line-file Composite style in a line + file shape <div><i></i> ROOT</div>

For reference, only one among A, SPAN and DIV may be used in a tag in the title area. 

<ul class="tree tree-style">
	<li class="open root">
		<i></i><a href="#">ROOT</a>
		<ul>
			<li class="open">
				List 1
			</li>
			<li class="fold">
				List 2
			</li>
			<li class="leaf last">
				List 3
			</li>
		</ul>
	</li>
</ul>
