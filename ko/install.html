Minify

JUI 라이브러리는 스타일만 별도로 사용할 수 있으며, 아래 두개의 CSS 파일을 로드하면 됩니다.
그리고 테마 파일은 다르게 사용할 수 있으며, 테마를 쉽게 만들 수 있는 별도의 툴을 제공할 예정입니다. 

<link rel="stylesheet" href="dist/jui.min.css" />
<link rel="stylesheet" href="dist/jennifer.theme.min.css" />

jui 클래스가 설정되어 있는 마크업의 하위에서만 사용할 수 있습니다. 

<body class="jui">
    ...
</body>

스크립트는 jQuery 1.8 이상에서만 동작하므로 해당 라이브러리를 먼저 로드해야 합니다. 

<script src="lib/jquery.min.js"></script>
<script src="dist/jui.min.js"></script>

Debug

스크립트는 디버깅 용도로 각각 별도로 로드할 수 있으며 base와 core는 필수 로드 파일입니다. 

<script src="js/base.js"></script>
<script src="js/core.js"></script>

그 다음에는 util과 ui 모듈을 먼저 로드하고, 순서대로 uix와 chart 모듈을 로드하면 됩니다. 

// base
<script src="js/base.js"></script>
<script src="js/core.js"></script>

// util
<script src="js/util/math.js"></script>
<script src="js/util/svg.js"></script>
...
// ui
<script src="js/ui/button.js"></script>
<script src="js/ui/combo.js"></script>
...
// uix
<script src="js/uix/autocomplete.js"></script>
<script src="js/uix/tab.js"></script>
...

// chart base
<script src="js/chart/draw.js"></script>
<script src="js/chart/builder.js"></script>

// chart theme
<script src="js/chart/theme/jennifer.js"></script>
<script src="js/chart/theme/dark.js"></script>
...
// chart grid
<script src="js/chart/grid/core.js"></script>
<script src="js/chart/grid/block.js"></script>
...
// chart brush
<script src="js/chart/brush/core.js"></script>
<script src="js/chart/brush/bar.js"></script>
...
// chart widget
<script src="js/chart/widget/core.js"></script>
<script src="js/chart/widget/tooltip.js"></script>
...

필요에 따라 사용하지 않는 모듈은 로드하지 않아도 되지만 종속 관계가 있는 모듈에서 에러가 발생할 수 있으니 유의하시기 바랍니다.

NodeJS

이제는 서버에서도 차트를 사용할 수 있습니다. 아래와 같이 npm 명령어를 사용해서 설치합니다. 

npm install jui

Build & Test

테스트를 수행하고 패키지에 포함된 .less와 .js 파일을 하나의 파일로 합치기 위해서는 아래와 같은 명령어를 실행하면 됩니다.
참고로 빌드 도구인 Grunt 모듈은 NodeJS가 설치되어 있어야 합니다. npm install grunt-cli -g
cd jui
npm install
grunt

빌드 및 테스트 명령어는 아래와 같이 기능 별로 따로 수행할 수 있습니다.

grunt js 명령어는 js 파일의 merge와 minify를 수행하고, grunt test는 컴포넌트의 테스트를 수행합니다.
또한 grunt css 명령어는 less 파일을 css로 변환하고, merge와 minify를 수행합니다. grunt js
grunt css
grunt test
