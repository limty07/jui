JENNIFER UI는 제니퍼소프트의 차기작인 HTML5 기반의 APM 솔루션인 JENNIFER5에서
사용되는 UI 컴포넌트를 정리한 라이브러리입니다.
JENNIFER UI 차트는 다양한 브러쉬와 그리드, 위젯을 제공하며
이를 조합하여 간단하고 쉽게 차트를 사용할 수 있습니다.
또한 기존의 테이블 컴포넌트와 연동하여 데이터를 표현할 수 있습니다.
JENNIFER UI는 정말 스마트한 라이브러리라 할 수 있죠.
Bootstrap 스타일에 JUI 스크립트를 적용해보세요.
다양하고 강력한 기능의 컴포넌트를 그대로 사용할 수 있습니다.
JENNIFER UI는 다양한 벡터 타입 아이콘을 제공합니다.
크기가 고정된 이미지가 아니기에 어떠한 크기로도 유연하게 사용할 수 있으며
고해상도 화면의 디바이스에서 더욱 빛을 발하게 될 것입니다.
